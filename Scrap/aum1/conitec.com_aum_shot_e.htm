Screenshot of the Month

    ACKNEX USERS MAGAZINE

presents screenshots from current ACKNEX games. As we will never be able to know all the ACKNEX games in all the world, we'll be glad to receive shots of any kind from you. You don't know how to do that? Well here is one way:

  • Start game in the DOS window under WIN95
  • Change to full screen mode with ALT+ENTER
  • Okay the WIN95 message "This application may only run in full screen mode" with OK
  • Click on frame selection in the menue bar of the window
  • Open frame within window and select the section that way
  • Click on "copy" in the menue bar of the window or push CTRL+C
  • Use ALT+ENTER to return to full screen mode
  • End application (WED) with F10 carry on at will
  • Start any paint program
  • Use "PASTE AS NEW" to insert the contents of the clipboard
  • Save file and edit at will

      Happy Shooting now!

      And now the screenshots of Monats!

Both screenshots this time originate from the 3D Adventure DER NAME DES BRUDERS, a politically engaged AntiFa-Game from Vienna. We are going to talk about this game in depth in the upcoming edition.

On the right you see an example for wall and floor textures fitting exactly as well as an excessive use of region ambients. In both cases the mood is just right. Keep up the good work!
