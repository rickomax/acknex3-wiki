Polygon Actors

Whow! Finally. The times are over when pixeled sprites plodded jerkily throught the levels. What had been a standard in games like QUAKE for quite a while has now finally been implemented in ACKNEX: actors made up of little polygons. Great! But how do you do it?

What we need are A) a model and B) a palette.

I won't expound on how to construct a model in this article. There are several tools they can be generated with. In the present case a simple geometrical figure was created using QMe. Having the right palette is the bigger problem. There are several tools for extracting a palette from an CPX file. One of them, PCX2PAL, you can download here. Follow the instructions of the DOS program you are given by simply invoking it. You'll wind up with a file with the extension ".pal", which you need to change into ".lmp" for QMe.
 

Once you have generated and saved your model (cf. QMe tutorial), you'll have a file with the extension ".mdl" which, like a bitmap, may be defined in the WDL like that:
 
  

    MODEL modernart1_mod <art1.mdl>;
So the model is defined like a BMAP.... 
    TEXTURE ModArt1_model_tex { 
 SCALE_XY    12,12;
 Model       modernart1_mod;
    }
...and is simply assigned to the texture. Afterwards we define it as an ACTOR or..
  
    THING ModernArt1_3D {TEXTURE ModArt1_model_tex;}
After that you only need to place it inside the level! Caution: if you're dowloading the test model, you should be aware that you'll most likely have the wrong palette.
 
 

How to generate a model in QMe: A short introduction.

Really only a short step-by-step introduction for generating a small test model. I won't say anything about working with the skin editor.
 
    1. After starting the program choose "New Model" from the file menue.

    2. Click on "Edit Frames" in the second icon board - a three-partite screen appears. In the large field you'll see a grid which you can move with both mouse buttons:

    3. Click left and hold - horizontal and vertical relocation of the object

    4. Click right and hold - rotating the object.

    5. Click "Geometry": a menue with several grid structures appears.

    6. Choose the model "Cone" in the menue "Geometry" - a cone appears in the centre of the grid.
     

Now you can play around with the object at your heart's delight, addingt further models to the present one, setting light sources, loading a skin, editing one or the other edge of the grid etc. Take your time and you'll learn how to generate a model. Or you won't and hire a commercial artist who knows how to do it!
