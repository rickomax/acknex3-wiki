# 3D GameStudio — ACKNEX Documentation (EN-US)

Welcome to the English (EN-US) documentation for **conitec 3D GameStudio**,
based on the **ACKNEX** 3‑D engine (reference manual version 3.9, May 1998,
and the bundled WDL tutorial).

3D GameStudio is a toolkit that lets you build 3‑D demos, role‑playing,
action, adventure and racing games — and publish them as standalone runtime
modules — without writing low‑level code. Games are described with the
**World Definition Language (WDL)** script and a **World Map (WMP)**
topography file, the latter normally produced with the **World Editor (WED)**.

> This wiki is a faithful transcription of the original conitec reference
> manual and tutorial. Some passages describe DOS / Windows 95 era behavior
> and hardware; they are preserved here for historical and reference accuracy.
